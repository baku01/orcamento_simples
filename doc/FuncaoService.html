Overview

Serviço para lidar com lógicas relacionadas à entidade Funcao.

Este módulo centraliza operações utilitárias e de negócio referentes à entidade Funcao, facilitando a manutenção e reutilização de código relacionado a funções do sistema.

Examples:

Buscar uma função por ID

funcao = FuncaoService.pega_funcao_por_id(1)

Buscar o valor base de uma função

valor_base = FuncaoService.pega_valor_base(funcao)
